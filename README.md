# silentvpn3.github.io

Лендинг Silent VPN — скачивание клиентов для Windows и Android.

**Сайт:** https://silentvpn3.github.io

Версии подтягиваются с production API; при сбое API — из `releases.json` и кеша браузера.

**После OTA-релиза** обновить `releases.json` (версия, size, filename) — fallback для страницы.
